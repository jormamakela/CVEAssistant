CVE Assistant

What is it?

CVE Assistant is a generalize assistant for CVE workflows. Having a generalized tool to help managing and reading Debian or other security vulnerabilities and mapping them against CVE information from NVD or other sources is difficult to find. During my work at Cisco Systems, I would've love to have such a tool available to keep better track of what was going on in the field. Hence, this tool.

This tool is also important to help people out there to have a relatively standard output to assist with bug reports and to fully understand the scope of work.

Why use this tool?

Is it free? Did it force me to write something from scratch again after many years to keep my skillset up? Did it solve my problem I was having? No one knows. If there is another set of tools out there with the exact same purpose, then great. This assistant is intended as a way to reinforce language learning for me, even as it's meant to be useful. The more tools the merrier. Really. This isn't intended to replace or solve issues that other tools may provide. Instead, it is more for myself and anyone else who might benefit from using it.
